# Ascend
Ascend is a game-playing agent model for the game "Ataxx" done for the "Nerdvana 2.0 Summer Camps 2026" highschool course by me, Flaviu Dimitriu.
This bot won the final tournament of this summer camp by winning 34 out of the 36 matches it played.
It uses alphaBeta algorithm with various optimizations to it.

To compile, use: "g++ -O2 -Wall -o ascend *.cpp" and this will create the binary file.

The protocol and arbiter for running games between programs of this kind can be found at https://github.com/nerdvana-ro/ataxx-tools.

Congrats to all participants at this camp and I must say that overall it was an awesome competition.
